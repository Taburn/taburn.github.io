Light Array 9

This page is under construction.

Light Array 9 is a 3x3 RGB LED display powered by USB Type-C. A button and knob on the back let the user change the displayed pattern and its speed (or other parameter). Its design focused on ease of manufacturing.

Motivation

  • Make something to give away at Christmas. This meant easy small batch production.
  • Gain more experience with product development. I designed the whole thing and had to solve the mechanical/electrical/ID conflicts.
  • Practice STM32 firmware development. This is the first non-trivial project where I've written all the code myself. It's also the first time I wrote a library.
  • Explore 3D modeling and printing. My company was kind enough to let me use their printers. Who would turn that down?

Schematic

The schematic is pretty straight forward. An STM32C011 is programmed via SWD to output colour information to a 3.3 to 5V level translator which sends it ot the daisy chained programmable RGB LEDs. A button is used to change the light pattern and a knob can control one of its parameters, usually speed. Power comes from a USB Type-C jack which provides 5V to the LEDs and a 3.3V linear regulator that poweres the microcontroller.

The STM32 uses its internal clock, so the only external components it needs is really just a 100nF cap and a programming header. The button has a bit of debouncing I found was necissary during testing and the pot has a probably unecissary low pass filter.

I picked the STM32C011J4M6 becasue there's soemthing really satisfying about using a low pin count µC and it was the easiest package to solder.

I wish I had picked the WS2812B instead of the WS2812C becasue it's a lot more popular. I'm planning to make a much larger display and can only find LED strips that use the B variant. It shouldn't be too much of an issue though, because it looks like I'll only have to adjust the bit timing parameter values.

PCB

My main goal for the PCB was to keep it to 2 layers while also providing a solid ground plane under every trace. I was mostly sucessful.
